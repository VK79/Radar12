Как получить Access Token
  1. Перейдите на страницу создания приложений
  2. Создайте Standalone-приложение
  3. Скопируйте ID приложения
  4. Перейдите по ссылке (замените CLIENT_ID):
    https://oauth.vk.com/authorize?client_id=CLIENT_ID&display=page&redirect_uri=https://oauth.vk.com/blank.html&scope=groups,wall,offline&response_type=token&v=5.131
  5. Разрешите доступ приложению
  6. Скопируйте access_token из адресной строки
