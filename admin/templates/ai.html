Что такое OpenRouter?

        
OpenRouter — это сервис, предоставляющий единый API для доступа к различным языковым моделям (GPT-4, Claude, Llama, DeepSeek и др.) с прозрачной оплатой.

        
        
Как получить API ключ?

        
    
            
  1. Зарегистрируйтесь на openrouter.ai
    2. 
            
  2. Пополните баланс (для платных моделей) или используйте бесплатные
    3. 
            
  3. Создайте API ключ в разделе Keys
    4. 
        

        
        
Бесплатные модели

        
Модели с суффиксом :free работают бесплатно, но могут иметь ограничения по скорости и количеству запросов.
